# Stargazer-test

## Requirement

|               | Version       |
| ------------- |:-------------:|
| NodeJS        | ^6            |
| MongoDB       | 3.4.4         |

## Usage

Start MongoDB
```
mongod
```

Start Api and App
```
cd api && npm install && npm start

cd app && npm install && npm start
```
